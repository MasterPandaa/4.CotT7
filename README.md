# Tetris dengan Pygame

Game Tetris sederhana berbasis Pygame. Mendukung kontrol dasar: kiri/kanan, rotasi, soft drop, dan hard drop.

## Fitur
- 10x20 grid playfield
- 7 tetromino klasik (I, O, T, S, Z, J, L)
- Rotasi dengan wall-kick sederhana
- Line clearing dan skor
- Peningkatan kecepatan jatuh seiring waktu

## Persyaratan
- Python 3.8+
- Pygame

## Instalasi
```
pip install -r requirements.txt
```

## Menjalankan
```
python tetris.py
```

## Kontrol
- Panah/Kiri atau A: geser ke kiri
- Panah/Kanan atau D: geser ke kanan
- Panah/Atas atau W: rotasi searah jarum jam
- Panah/Bawah atau S: soft drop
- Spasi: hard drop
- ESC: keluar
